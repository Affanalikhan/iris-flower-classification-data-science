## 🎯 Overview

A clean, well-documented classification project on the Iris dataset. Classifies flowers into three species — *Setosa*, *Versicolor*, and *Virginica* — based on sepal and petal measurements. Great for understanding classification fundamentals, model comparison, and visualization techniques.

---

## 🌺 Species Classified

| Species | Characteristics |
|---|---|
| Iris Setosa | Smallest petals, most separable |
| Iris Versicolor | Medium size, some overlap |
| Iris Virginica | Largest petals, overlaps with Versicolor |

---

## 🔍 What's Inside

- **EDA** — Pairplots, violin plots, correlation heatmap
- **Models** — KNN, Decision Tree, SVM, Logistic Regression, Random Forest
- **Evaluation** — Accuracy, confusion matrix, classification report
- **Visualization** — Decision boundaries and feature importance
